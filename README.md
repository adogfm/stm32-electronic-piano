# STM32 嵌入式电子琴交互系统

基于 STM32F103C8T6 的电子琴交互系统，支持 16 键触摸演奏、实时录音回放、多音色切换与自动伴奏，为音乐爱好者与嵌入式学习者提供开箱即用的电子乐器方案。

> 项目已完整调试验证，包含完整源码、硬件驱动及演示曲谱。


## 技术栈

- **主控芯片**：STM32F103C8T6（Cortex-M3，72MHz）
- **开发环境**：Keil uVision5 + STM32 标准外设库
- **编程语言**：C 语言（裸机开发，无操作系统）
- **外设驱动**：GPIO、定时器（TIM2/TIM3）、PWM、SPI、EXTI
- **交互模块**：TTP229 电容触摸键盘（16 键）、0.96 寸 OLED（SSD1306，SPI 接口）、WS2812 RGB 灯带（4 灯）
- **音频输出**：无源蜂鸣器 + 双通道 PWM 方波合成


## 核心功能

- **实时演奏**：16 键触摸感应，支持多指同时触控，响应延迟 < 10ms
- **双通道音频合成**：旋律通道 + 伴奏通道独立发声，支持 4 种音色（大提琴 / 电风琴 / 吉他 / 打击乐）和 4 种音调模式（正常 / 升调 / 降调 / 高八度）
- **录音与回放**：支持实时录制演奏片段（最多 128 个音符），一键回放
- **自动伴奏模式**：内置《小星星》曲谱，节拍 LED 同步闪烁，辅助练习与演示
- **可视化交互反馈**：OLED 实时显示当前模式、音色、音调及按键状态；WS2812 RGB 灯带根据按键动态变色


## 硬件连接

| 模块 | 引脚 | 说明 |
|------|------|------|
| TTP229 触摸键盘 | SCL→PB6, SDO→PB7 | 16 键电容触摸 |
| OLED 屏幕 | CS→PB11, DC→PB12, RES→PB14, SCK→PB13, MOSI→PB15 | SPI 接口 |
| WS2812 灯带 | DIN→PB10 | 4 颗 RGB LED |
| 蜂鸣器（旋律） | PA0 | PWM 通道 1 |
| 蜂鸣器（伴奏） | PA7 | PWM 通道 2 |
| 按键 1/2/3 | PA5/PA8/PA10 | 切换音色/音调/模式 |

### 环境准备
- Keil uVision5（支持 STM32F10x 系列）
- ST-Link 下载器（或其他 ARM 调试器）
- 硬件按上表连接

### 编译与烧录
1. 克隆项目
```bash
git clone https://github.com/adogfm/stm32-electronic-piano.git
cd stm32-electronic-piano
```

2. 用 Keil 打开项目工程文件（`.uvprojx`）

3. 编译并下载到 STM32F103C8T6

4. 上电后 OLED 显示 "Piano Ready"，触摸键盘即可演奏


## 项目结构

```
stm32-electronic-piano/
├── main.c               # 主程序（模式管理、状态机）
├── Buzzer.c/h           # 双通道 PWM 音频驱动
├── PWM.c/h              # 定时器 PWM 配置
├── OLED.c/h             # 0.96 寸 OLED 显示驱动
├── TouchKey.c/h         # TTP229 触摸键盘驱动
├── LED.c/h              # 指示灯控制
├── Key.c/h              # 物理按键驱动
├── Encoder.c/h          # 编码器驱动（扩展）
├── Delay.c/h            # 延时函数
└── stm32f10x_conf.h     # 标准外设库配置
```


## 项目亮点

- **裸机多任务调度**：10ms 主循环定时器架构，实现按键扫描、音频合成、LED 刷新、节拍生成的多任务实时调度
- **软件防抖与误触抑制**：通过双重触摸读取校验和软件消抖算法，有效解决电容触摸误触发问题
- **完整的嵌入式工程结构**：硬件抽象层（HAL-style）分层设计，驱动与业务逻辑解耦
- **可扩展性**：预留编码器接口和打击乐通道，便于后续功能扩展
